Working with Sassle

Working with Sassle is just like working on any other Django project except you can now use compass to compile SASS files from your media/sass directory. Sassle uses a common sense naming and flexible inclusion conventions to help you start building maintainable projects quickly.

Sassle is made possible by the following projects:

  • Compass
  • SASS
  • Django
